# jsguide

Dynamic forms, from json data to angular views.


## Usage

To install this component in your current working directory:

```shell
bower install --save 2sol/formie
```


## For Developers


### Building

```shell
npm install
bower install
gulp build
```


### Interactive demo

```shell
cd demo
bower install
cd ..
gulp demo
```


### Testing

```shell
gulp test
```
